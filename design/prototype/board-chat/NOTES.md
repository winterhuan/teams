## 待验证问题

1. **Board 落地是否比旧 Home 顺？** 新版本/硬门塌成角标够不够醒目，还是需要更强提示？
2. **点卡进 Chat 的过渡** 是否有"拉近"感，还是像跳了一层页？
3. **活产物右面板** 只在有产物时出现——空/满切换是否突兀？
4. **无产物 issue 整屏对话** 是否舒服，还是空得慌？
5. **In Review 跨模型互审** 语义在 UI 上够不够清楚（谁审谁、为什么是另一模型族）？
6. **Inbox 抽屉** vs 直接在 board 上看角标——两者会不会重复/打架？
7. **inbox project 兜底** 的表达（虚线卡）够不够自然？

- Board 拖拽改状态未做
